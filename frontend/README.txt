0. 필수 설치*node, npm 설치 터미널 명령어node -vnpm -v*의존성 설치 - "프로젝트 루트(cd faith-front)"에서npm install*개발 서버 실행 명령어npm run dev*vite 프로젝트 만들어야 한다고 뜨는 경우npm install 꼭 하기 or 현재 파일경로가 faith-front 루트가 맞는지 확인하기---------------------------------------------------1. 스택 정리프레임워크: React 18번들러: Vite스타일링: TailwindCSS 4.1.x아이콘: lucide-reactState: React useState / useRefAuth 방식: JWT---------------------------------------------------2. 이 faith-front 디렉토리 구조 그대로 사용하기components/: 재사용 UI 컴포넌트views: 페이지 단위 View.→ Home은 메인홈, About은 서비스 소개, Login은 로그인 폼lib/: 비즈니스 로직 / 유틸→ validators.js: 텍스트 입력할때 5자 이상으로 입력하는 조건 검수auth.js: JWT 저장/조회/삭제 임시로 만들어둠api/: API 함수 껍데기 저장해둔것들. 실제 응답 스키마는 백엔드 기준으로 처리---------------------------------------------------3. 백엔드에서 사용하는 필드명 오류/다름/변경 시 알려주세요.4. !TailwindCSS 주의사항... 버전 충돌 오류 주의tailwind.config.js 파일 없는게 맞음위의 config 파일을 추가하거나 버전을 V3으로 바꿔서 코드 수정하면 오류 발생함
